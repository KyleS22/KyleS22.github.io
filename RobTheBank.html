Rob The Bank

Rob The Bank is a game I helped develop for the 2016 Game Jam. We designed and developed the entire game in 48 hours. I worked with one other person. For this game, I created the art and sound effects, as well as designed the levels. It is a simple game, but for two people and 48 hours, it was quite a bit of work.

The game was developed in the Unity game editor. If you would like to play the game, instructions to download and play the game are on the official game jam website. If you would like to learn more about the game jam, visit www.gamewithus.ca




Tools Used

  • Unity 3D
  • C#
  • Adobe Illustrator
  • LMMS

My Contributions

  • Desgined and created sprites for the Game
  • Designed levels for the game
  • Implementation of game logic
  • Created a soundtrack
